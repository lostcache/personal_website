# Technology Stack

## Vision

Migrate from the existing Astro framework to a zero-dependency architecture.

## Core Technologies

- **System/Backend:** C (Custom implementation).
- **Frontend Structure:** HTML5 (Vanilla).
- **Frontend Styling:** CSS3 (Vanilla).
- **Frontend Logic:** JavaScript (Vanilla, no frameworks).

## Constraints

- **Zero Dependencies:** No external libraries or frameworks (e.g., no React, no Astro, no npm packages).
- **Performance:** Optimized for raw speed and minimal footprint.
